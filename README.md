# Student Grade Management System (C++)

This project is developed in C++ using Object-Oriented Programming concepts.

## Features
- Store student personal information
- Input marks with validation
- Calculate total marks
- Calculate percentage
- Generate grades automatically
- Display failed subjects

## Concepts Used
- Classes and Objects
- Multiple Inheritance
- Arrays
- Functions
- Loops
- Conditional Statements

## Technologies
- C++
- OOP

## Output
The program displays:
- Student details
- Subject marks
- Grades
- Percentage
- Failed subjects (if any)
